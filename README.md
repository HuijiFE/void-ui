# Void-UI Controls

Work in progress...

## Development

### Environment

- Node.js: >= 8, recommend to use the latest version.
- Yarn: package manager, for install dependencies.

### Steps

```
# clone the project
git clone git@github.com:HuijiFE/void-ui.git

cd void-ui

# install all dependencies
yarn install

# run the server
yarn serve
```
